# SafeTube Releases

Welcome to the official releases repository for SafeTube - a family-friendly YouTube content management application.

## About SafeTube

SafeTube is designed to provide a safer YouTube experience for children and families by offering content filtering, whitelisting, and parental controls.

## Releases

This repository hosts binary releases and version information for SafeTube. You can find the latest releases in the [Releases](../../releases) section.

### Latest Release

To download the latest version of SafeTube, visit the [Releases page](../../releases/latest).

### Supported Platforms

- Windows (x64)
- macOS (Intel & Apple Silicon)
- Linux (x64)

## Installation

1. Navigate to the [Releases](../../releases) page
2. Download the appropriate installer for your platform
3. Run the installer and follow the on-screen instructions
4. Launch SafeTube and configure your parental controls
